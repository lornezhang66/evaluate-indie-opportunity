# 证据与评分规则

## 证据等级

使用当前机会达到的最高等级。不要把多个低等级信号相加，伪装成更高等级证据。

| 等级 | 含义 | 可接受的典型证据 | 最终分数上限 |
|---|---|---|---:|
| E0 | 灵感 | 个人直觉、媒体故事、榜单、仓库介绍、别人赚钱案例 | 49 |
| E1 | 市场代理 | 当前竞品、搜索趋势、公开抱怨、Issue、用户自建 workaround | 59 |
| E2 | 行为证据 | 目标用户访谈、候补名单、自然下载、重复使用、主动咨询 | 74 |
| E3 | 承诺证据 | 预付、押金、付费试点、签署意向、真实付款、明确资源承诺 | 89 |
| E4 | 重复证据 | 复购、留存、转介绍、稳定获客、可重复成交或单位经济成立 | 100 |

注意：

- 点赞、收藏、“我会用”和免费注册通常不高于 E2。
- 下载量是采用信号，不是留存或付款信号。
- Star、热门技术和“可做 SaaS/API/咨询”通常只属于 E0–E1。
- 企业口头认可不等于采购承诺。
- 创始人自己每天使用是有价值的行为证据，但不能替代外部需求。

## 八维评分表

每项先给 0–5 分，再乘权重。总分 100。

| 维度 | 权重 | 0 分 | 3 分 | 5 分 |
|---|---:|---|---|---|
| 付费痛点与紧迫性 | ×4 | 没有具体痛点或付费者 | 问题明确但可拖延 | 高频、高损且已有预算或替代支出 |
| 证据强度 | ×4 | 只有想象 | 有多项 E1/E2 信号 | 已有 E3/E4 证据 |
| 结构化优势 | ×3 | 与任何人无差别 | 有技能、经历或自用优势 | 独特领域经验、渠道、数据或可信身份形成组合优势 |
| 触达与分发 | ×3 | 不知道去哪里找用户 | 有可执行渠道但尚未验证 | 已拥有稳定触达或平台自然分发 |
| 第一笔交易清晰度 | ×2 | 只能描述未来规模 | 能说清报价和候选人 | 已有具体买家、报价和承诺路径 |
| Phase 0 效率 | ×2 | 必须先做完整产品 | 两到六周可验证 | 数天到两周内可低成本证伪 |
| 复利与留存 | ×1 | 完全按工时出售 | 部分可复用 | 使用、内容、代码、数据、口碑或渠道持续累积 |
| 可持续与边界 | ×1 | 明显越界或长期透支 | 风险可管理 | 与本职/IP 清晰隔离且维护负担适配 |

计算：

`潜力分 = Σ(单项 0–5 分 × 权重)`

`最终分 = min(潜力分, 当前证据等级上限)`

不要给小数。若同一风险已经导致红线结论，不再通过低分重复“稀释”它。

## 决策规则

| 条件 | 默认动作 |
|---|---|
| 存在伦理、合法性或不可接受的 IP/保密红线 | AVOID / BOUNDARY REVIEW |
| E4 且最终分 ≥ 80 | COMMIT |
| E3 且最终分 ≥ 70 | COMMIT 或扩大付费验证 |
| E2 且最终分 ≥ 60 | TEST NOW |
| E0–E1 且潜力分较高 | RESEARCH；先补证据，不因高潜力分开工 |
| E0–E1，但 Phase 0 不超过两周、损失封顶且不挤占主线 | 可 TEST NOW，但只把它定义为现金或学习实验，不得升级为产品承诺 |
| 最终分 40–59 | RESEARCH、WATCH 或 PARK |
| 最终分 < 40 | PARK；除非它是成本极低的战略学习期权 |

分数接近时，按以下顺序破局：

1. 更接近付款；
2. 更容易触达；
3. 更能复用现有资产；
4. 更快被证伪；
5. 更少侵占正式开发主线。

## 置信度

- **低**：核心判断主要依赖假设或过时资料。
- **中**：有当前外部信号，但付款、留存或分发仍未验证。
- **高**：有多来源事实以及真实承诺、重复行为或交易数据。
